Day 0｜7/23 台中 → 桃園機場過夜
前一天晚上出發，降低凌晨趕車壓力。
台中火車站 → 客運 → 桃園機場 T2 → 5F 美食街外圍休息
🚍 交通與出口建議
台中火車站／台中轉運站 → 國光或統聯客運 → 桃園機場第二航廈。抵達後依 T2「5F 美食街」指標。
Day 1｜7/24 神戶牛＋大阪天神祭宵宮
神戶進、大阪住，第一天不塞太滿。
桃園 T2 → 神戶機場 → 三宮 → Mouriya 本店 → APA 天滿橋 → 天神祭宵宮
🚃 交通與出口建議
神戶機場站 → Port Liner → 三宮站；三宮往西側／北野坂方向到 Mouriya。往大阪住宿可轉 Osaka Metro 谷町線至天滿橋站，4 號出口較順。
Day 2｜7/25 童書之森＋天神祭本宮
白天室內輕鬆，晚上看祭典。
APA天滿橋 → 童書之森中之島 → 北浜午餐 → 飯店午休 → 天滿橋觀賞點
🚇 交通與出口建議
京阪中之島線「なにわ橋站」1 號出口最接近中之島公園與童書之森。晚上回天滿橋站，往大川河岸方向出口。
Day 3｜7/26 海遊館主軸＋移動到 USJ 區
一個主景點，不消耗隔天 USJ 體力。
天滿橋 → 海遊館 → 天保山午餐 → Liber Hotel
🚇 交通與出口建議
Osaka Metro 中央線「大阪港站」1 號出口最接近海遊館。往飯店可轉 JR 櫻島線至「櫻島站」。
Day 4｜7/27 USJ 環球影城
早上熱門、下午表演與輕鬆設施。
Liber → USJ入口 → 哈利波特/任天堂 → 蘑菇餐廳 → 水世界 → CityWalk
🚶 交通與出口建議
Liber Hotel 可步行或搭 JR 至「環球城站」。出站後穿過 Universal CityWalk，跟著 USJ 入口指標。
打開 USJ 專屬攻略
Day 5｜7/28 換飯店＋心齋橋購物恢復日
USJ隔天恢復體力，慢慢逛。
Liber → OASIS NANIWA → 難波午餐 → シモジマ → 道頓堀
🚃 交通與出口建議
JR 櫻島線 → 西九條 → 阪神難波線 → 大阪難波。心齋橋站 5、6 號出口往商店街較順；道頓堀可用難波站 14、15-B 方向。
Day 6｜7/29 京都 teamLab
一主一副，看光影、走河邊，不暴走京都。
大阪 → 京都車站 → 拉麵小路 → teamLab Biovortex Kyoto → 鴨川或祇園 → 回大阪
🚄 交通與出口建議
大阪站 → JR 京都線 → 京都站。前往 teamLab 建議從京都站八條東口／東南側方向；祇園可用京阪祇園四条站 6、7 號出口。
Day 7｜7/30 自由購物＋補買日
整天保留彈性。
OASIS → Joshin Super Kids Land → 黑門市場／難波補買 → 飯店整理行李
🚇 交通與出口建議
黑門市場可用日本橋站 10 號出口方向。Joshin Super Kids Land 可從難波往日本橋、電電城方向步行。
Day 8｜7/31 回程日
早上直接前往關西機場。
飯店 → 南海難波站 → 南海電鐵 Rapi:t／空港急行 → 關西機場 T1
✈️ 交通與出口建議
南海難波站 → 關西機場站。出閘後依 Terminal 1／國際線出發指標前進。BR177 11:10 起飛，09:00 前到機場最穩。
