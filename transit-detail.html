Day 0｜台中 → 桃園機場 T2
機場過夜
台中火車站／台中轉運站 → 國光或統聯客運 → 桃園國際機場第二航廈
上車點
台中火車站周邊客運站，出發前確認實際月台與候車處。
下車點
桃園機場第二航廈。
最近出口／最快方向
抵達 T2 後依「5F 美食街」指標上樓，休息區較集中。
Day 1｜神戶機場 → 三宮 → 天滿橋
神戶進大阪住
神戶機場站 → Port Liner → 三宮站 → Mouriya 本店 → JR／阪神轉 Osaka Metro 谷町線 → 天滿橋站
神戶機場到三宮
Port Liner 神戶新交通神戶機場站搭到三宮站。
Mouriya 本店
從三宮站往西側、北野坂／生田新道方向步行較順。
前往 APA 天滿橋
由三宮進大阪市區後，轉 Osaka Metro 谷町線至「天滿橋站」。
最近出口：天滿橋站 4 號出口，或依 Google Maps 即時出口。
晚上天神祭：回到天滿橋站後往大川河岸方向，不建議直接擠橋中央。
Day 2｜天滿橋 → 中之島 → 天神祭
祭典日
天滿橋站 → 京阪中之島線／步行銜接 → なにわ橋站 → 童書之森中之島 → 北浜 → 天滿橋
童書之森中之島
建議搭至「なにわ橋站」。
最近出口：なにわ橋站 1 號出口，往中之島公園與童書之森方向。
北浜午餐
可由中之島步行至北浜河岸咖啡一帶。
天神祭觀賞
回「天滿橋站」，沿河岸找可撤退的位置。
Day 3｜天滿橋 → 海遊館 → Liber Hotel
海遊館主軸
天滿橋站 → Osaka Metro 谷町線轉中央線 → 大阪港站 → 海遊館 → 弁天町／西九條 → JR 櫻島線 → 櫻島站
海遊館最近站
Osaka Metro 中央線大阪港站。
最近出口
大阪港站 1 號出口最接近海遊館與天保山 Market Place。
前往 Liber Hotel
大阪港站轉往弁天町／西九條，再搭 JR 櫻島線到「櫻島站」。櫻島站出站後依 Liber Hotel 指標，步行距離短。
Day 4｜Liber Hotel → USJ
USJ 重點日
Liber Hotel → 步行或 JR 櫻島線 → 環球城站 → USJ 入口
最近站
JR 夢咲線／櫻島線「環球城站」。
最近出口／最快方向
環球城站出站後穿過 Universal CityWalk，跟著 USJ 入口指標前進。
小提醒
早上不要先逛商店，先往入園口排隊。
Day 5｜USJ 區 → OASIS NANIWA → 心齋橋／道頓堀
恢復購物日
櫻島站／環球城站 → JR 櫻島線 → 西九條站 → 阪神難波線 → 大阪難波站 → OASIS NANIWA
換飯店
USJ 區先到西九條，再接阪神難波線到大阪難波。
心齋橋購物
Osaka Metro 御堂筋線心齋橋站。
最近出口：心齋橋站 5、6 號出口往商店街較順。
道頓堀
可從難波站 14、15-B 一帶出口步行。
Day 6｜大阪 → 京都 teamLab
京都療癒日
大阪站／難波周邊 → JR 京都線或阪急京都線 → 京都站／河原町 → teamLab Biovortex Kyoto → 鴨川或祇園 → 回大阪
省力路線
JR 京都線到「京都站」，午餐直接在京都站拉麵小路。
teamLab
從京都站八條東口／東南側方向步行或短程計程車較省力。
最近出口：京都站八條東口。
祇園／鴨川
可搭公車或計程車至祇園四条、河原町一帶。
京阪「祇園四条站」6、7 號出口。
Day 7｜自由購物與補買
彈性日
OASIS NANIWA → 難波站／心齋橋站 → 道頓堀／藥妝／百貨 → 回住宿
難波商圈
Osaka Metro 御堂筋線「難波站」或「心齋橋站」。
出口提醒
道頓堀用難波站 14、15-B 方向；心齋橋筋用心齋橋站 5、6 號出口。
小提醒
購物日保留回飯店整理行李時間。
Day 8｜大阪 → 關西機場 T1
回程日
難波站 → 南海電鐵 Rapi:t／空港急行 → 關西機場站 → T1
建議路線
南海電鐵 Rapi:t／空港急行南海難波站搭到「關西機場站」。
最近出口／最快方向
關西機場站出閘後依「Terminal 1／國際線出發」指標前進。
航班提醒
BR177：11:10 起飛，建議 09:00 前抵達關西機場 T1。
